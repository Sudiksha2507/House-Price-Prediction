# House Price Prediction using Machine Learning

## Overview

This project predicts house prices based on house characteristics such as living area, overall quality, garage capacity, basement area, and bathrooms.

## Dataset

Kaggle House Prices - Advanced Regression Techniques Dataset.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

## Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Training
6. Model Evaluation
7. House Price Prediction

## Models Used

* Linear Regression
* Random Forest Regressor

## Evaluation Metrics

* R² Score
* Mean Absolute Error (MAE)
