# House Price Prediction

## Overview
This machine learning project predicts house prices using house features such as area, quality, garage capacity, basement area, and bathrooms.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Google Colab

## Models
- Linear Regression
- Random Forest Regressor

## Results
R² Score: 0.90

## Dataset
Kaggle House Prices Dataset
